The project is organized into dedicated folders, each focusing on a specific Java pillar:

* **`classes (entry level)`**: Basics of class structure, object instantiation, and method definitions.
* **`Encapculation_example`**: Implementation of data hiding using private fields and getter/setter methods.
* **`inheritance` & `inheritance_2`**: Examples showcasing class hierarchies, the `super` keyword, and code reusability.
* **`statics`**: Practical usage of the `static` keyword for memory management and utility methods.
* **`final_samples`**: Demonstrations of constant variables, non-overridable methods, and non-inheritable classes.
* **`Examples`**: General practice logic, algorithmic exercises, and miscellaneous Java samples.
